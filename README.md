# Chat
Project 1 - OS

![demo](https://github.com/gusmendez99/os-chat/raw/main/images/demo.gif?raw=true)

### Install

Please, be sure to install before Ncurses and Protobuf in your environment.

### Usage
```shell
make
```  
#### Server
```shell
./server <port>
```  

#### Client
```shell
./client <username> <server-ip> <server-port>
```
